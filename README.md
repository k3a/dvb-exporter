# DVB Exporter

A simple exporter to export basic DVB subsystem statistics in Prometheus format.

It exports statistics from the V3 API (like `femon`) as well as additional stats from the V5 API (SNR and signal strength in dBs, pre/post FEC errors, and total bit counts).

# Contribute
I don't expect to add much more functionality to it.
If you find something missing or an error, feel free to create a pull request.
